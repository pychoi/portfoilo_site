Grace's Kitchen

Visit Site

Grace's Kitchen is a web application I developed for my family and friends to share, store, and search recipes and generating groceries lists.

Features/Functionality

  • Random featured recipe
  • Recipe search functionality
  • Add Recipes
  • Save Recipes / Weekly meal plan list management
  • Groceries list generator

Technologies Used

MongoDB, Mongoose, Node.js, Express.js, AngularJS, Passport.js, Angular Material, HTML, CSS
